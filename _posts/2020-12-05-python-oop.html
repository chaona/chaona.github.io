构造函数

可以看成一个特殊的实例方法。调用方式和意义不一样。


class Student():
	# 特征，类变量
	name = ''
	age = 0

	# 构造函数,实例化过程中，就会被调用。
	# 构造函数，只能返回 None
	# 作用：可以生产不同的对象
	def __init__(self, name, age): # self 显胜于隐；self和对象有关，代表的是实例；self必须写
		# 初始化对象的属性
		# 实例变量
		self.name = name
		self.age = age
		# print('student')

	# 行为
	def do_homework(self):
		print('homework')

类变量与实例变量

类变量就是和类相关联的变量。实例变量就是和对象相关的变量。

类与对象的变量查找顺序

如果实例变量不存在，它会在类变量里面找。如果类里没有，又会从父类里找。

self与实例方法

实例方法定义的时候，必须有self。调用时，不需要。

self 显胜于隐；self和对象有关，代表的是实例。

self也可以用别的名字代替，所以叫它关键字不合适。

在实例方法中访问实例变量与类变量


class Student():
	# 一个班级的所有学生
	sum = 0
	# 特征
	name = ''
	age = 0

	# 构造函数,实例化过程中，就会被调用。
	# 构造函数，只能返回 None
	# 作用：可以生产不同的对象
	def __init__(self, name, age):
		# 初始化对象的属性
		self.name = name
		self.age = age
		# print(Student.sum1)
		self.__class__.sum += 1
		print('当前学生的总数： ' + str(self.__class__.sum))

	# 行为
	def do_homework(self):
		print('homework')

类方法

类方法：操作和类相关的变量。


class Student():
	# 一个班级的所有学生
	num = 0
	# 特征
	name = ''
	age = 0

	# 构造函数,实例化过程中，就会被调用。
	# 构造函数，只能返回 None
	# 作用：可以生产不同的对象
	def __init__(self, name, age):
		# 初始化对象的属性
		self.name = name
		self.age = age

	# 实例方法 行为
	def do_homework(self):
		print('homework')

	# 类方法 @装饰器；从规范的角度，建议使用类来调用
	@classmethod
	def plus_sum(cls):
		cls.num += 1
		print('当前学生的总数： ' + str(cls.num))
