在Python中变量就是“名字”。命名可读性要强！

七种运算符：算术运算符、赋值运算符、比较（关系）运算符、逻辑运算符、成员运算符、身份运算符、位运算符。

1、变量

变量就是名字。

变量的命名规则：字母、数字、下划线。区分大小写。数字不能打头。系统关键字（保留关键字）不能用作变量名。


>>> a = 1
>>> b = a
>>> a = 3
>>> print(b)
1
>>> a = [1,2,3,4,5]
>>> b = a
>>> a[0] = '1'
>>> print(a)
['1', 2, 3, 4, 5]
>>> print(b)
['1', 2, 3, 4, 5]

3、set 集合

集合是无序，不重复

>>> {1,2,4,3,4,3,2,1,1}
{1, 2, 3, 4}
>>> len({1,2,3,4,5})
5
>>> 1 in {1,2,3}
True
>>> 1 not in {1,2,3}
False
>>> {1,2,3,4,5,6,7} - {5,6}
{1, 2, 3, 4, 7}
>>> {1,2,3,4,5,6,7} - {5,6,8}
{1, 2, 3, 4, 7}
>>> {1,2,3,4} & {3,4}
{3, 4}
>>> {1,2,3,4,5,6} | {3,4,7}
{1, 2, 3, 4, 5, 6, 7}
>>> type(set())
class 'set'
>>> len(set())
0

“-”表示求两个集合的差集。“&”表示交集。“|”表示并集。使用set()来定义空集。

4、dict 字典

key和value,最常见的操作是通过key找value

一个字典有很多个key和value构成，是集合类型（set）。set没有key，dict有。

value可以是int str float list set dict 等等任意类型。

key只能是不可变类型：int str tuple

>>> type({'q':'query','w':'world','h':'hello'})
class 'dict'
>>> {'q':'query','w':'world','h':'hello'}['q']
'query'
>>> {'q':'query','w':'world','h':'hello'}['h']
'hello'
