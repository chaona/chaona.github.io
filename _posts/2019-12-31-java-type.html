声明变量

变量就像是杯子，是一种容器，承载某些事物。它有大小与类型。

Java注重类型。

声明变量的规则：

  1. 变量必须拥有类型。
  2. 变量必须要有名称。
//类型 名称
int count;

primitive主要数据类型有八种

六种数字类型（四个整数型，两个浮点型），一种字符类型，还有一种布尔型。

  • 数字类型:
    • byte 8
    • short 16
    • int 32
    • long 64
    • float 32
    • double 64
  • char 16
  • boolean

对象引用

  • 事实上没有对象变量这样的东西存在。
  • 只有引用（reference）到对象的变量。
  • 对象引用变量保存的是存取对象的方法。
  • 它并不是对象的容器，而是类似指向对象的指针。

数组

数组也是对象。

// 声明一个int数字变量。数组变量是数组对象的遥控器。
int[] nums;

// 创建大小为7的数组，并将它赋值给之前声明为int[]的变量nums。
nums = new int[7];

// 赋予int数组的每一个元素一个int值。
nums[0] = 6;
nums[1] = 19;
nums[2] = 44;
nums[3] = 42;
nums[4] = 10;
nums[5] = 20;
nums[6] = 1;

创建Dog数组

// 声明一个Dog类
class Dog {
	String name;
	public static void main(String[] args){
		// create Dog object
		Dog dog1 = new Dog();
		dog1.bark();
		dog1.name = "Bart";

		// creat Dog array
		Dog[] myDogs = new Dog[3];

		// close door
		myDogs[0] = new Dog();
		myDogs[1] = new Dog();
		myDogs[2] = dog1;

		// set Dog by arr reference 
		myDogs[0].name = "Fred";
		myDogs[1].name = "Marge";

		// print
		System.out.print("last dog's name is ");
		System.out.println(myDogs[2].name);

		// Doy do bark() one by one
		int x = 0;
		while(x < myDogs.length){
			myDogs[x].bark();
			x = x + 1;
		}
	}

	public void bark() {
		System.out.println(name + " says Ruff!");
	}
}

要点

  • 变量有两种:primitive主要数据类型和引用
  • 变量的声明必须有类型和名称
  • primitive主要数据类型变量值是该值得字节所表示的.
  • 引用变量的值代表位于堆之对象的存取方法.
  • 引用变量如同遥控器,对引用变量使用圆点运算符"."可以如同按下遥控器按钮般di存取它的方法或实例变量.
  • 没有引用到任何对象的引用变量的值为null值.
  • 数组一定是个对象,不管所声明的元素是否为primitive主要数据类型.
