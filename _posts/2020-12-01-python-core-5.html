函数的定义及运行特点

1. 参数列表可以没有。
2.可以有return value,没有的话，默认返回None。

'''
1. 实现两个数字的相加
2. 打印输入的参数
'''

def add(a, b):
	result = a + b
	return result

def print_code(code):
	print(code)

x = add(1, 2)
y = print_code('Python')

print(x, y)

如何让函数返回多个结果

'''
计算英雄技能的伤害 
SF skill1 skill2 skill3
'''

def damage(skill1, skill2, skill3):
	damage1 = skill1 * 3
	damage2 = skill2 * 2 + 10
	damage3 = skill3 * 5
	return damage1, damage2, damage3

result = damage(1,2,3)
print(type(result)) # class 'tuple'

#序列解包 tuple元组；变量的命名要有意义
skill1_damage, skill2_damage, skill3_damage = damage(1,2,3)

print(skill1_damage, skill2_damage, skill3_damage)
#3 14 15
